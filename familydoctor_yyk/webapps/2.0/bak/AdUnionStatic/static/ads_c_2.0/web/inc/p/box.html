信息提示：

我们为您提供了精确匹配、短语匹配、广泛匹配三种不同类型的关键词匹配方式，使用不同的匹配方式将帮助您通过不同的方式定位潜在客户，您可以通过特殊符号标示关键词的匹配模式。关键词 = 广泛匹配：针对关键词的相关搜索词自动展现您的创意； "关键词" = 短语匹配：只在用户搜索词与关键词字面一致、或者搜索词包含关键词时才展现对应的创意；[关键词] = 精确匹配：用户搜索词与关键词字面完全一致时才展现对应的创意。

量身推荐：

针对具体的推广计划或推广单元的业务内容，为您提供量身定制的推荐关键词。

日均搜索量：

最近一周内网民的日均搜索次数或者范围。日均搜索量大小仅与关键词本身、推广地域有关系，一般来说，推广地域越多，同一个关键词的日均搜索量越大。

竞争激烈程度：

数字越大，代表该关键词被越多的同行所关注。此外，竞争激烈程度还与您选择的具体地域以及该关键词在不同匹配方式下，竞争者的数量有关。
